# Calculador-Transporte-Sustentavel
Compara a sustentabilidade dos principais meios de transportes do mundo.

 Sobre o Projeto:

O sistema permite visualizar e comparar diferentes meios de transporte como:

- Avião
- Carro
- Moto
- Ônibus
- Bicicleta

O foco principal é demonstrar, de forma visual e organizada, qual opção é mais sustentável dependendo do contexto.


 Tecnologias Utilizadas:

- HTML5
- CSS3
- Organização em pastas (HTML, CSS e imagens)

  Estrutura do Projeto:

Comparador-Transporte-Sustentavel/
│
├── HTML/
├── CSS/
├── imagens/
└── README.md



 Objetivo:

Este projeto foi desenvolvido como prática de organização de código, estruturação de páginas e estilização com CSS, com foco em boas práticas de desenvolvimento front-end.
